Лабораторная работа: Источники задач и контракты
Целью лабораторной работы является изучение принципов построения абстракций и работы с источниками данных.
В рамках задания требовалось реализовать источник задач с использованием заданного интерфейса, а также обеспечить получение данных из внешнего источника.
На начальном этапе был реализован источник задач с использованием API-заглушки, в котором данные формировались вручную внутри класса.
В процессе выполнения работы заглушка была заменена на обращение к реальному API. Для этого был реализован HTTP-запрос к внешнему сервису с использованием библиотеки requests. Полученные данные в формате JSON обрабатываются и преобразуются в объекты класса Task.
В результате был реализован класс ApiTaskSource, обеспечивающий получение данных из внешнего API, их обработку и возврат в виде списка задач.
Дополнительно реализована обработка возможных ошибок при выполнении HTTP-запроса.
Результат работы: разработан источник задач, интегрированный с внешним API и соответствующий заданному интерфейсу.
